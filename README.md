### CN2 Pin Descriptions:
This connector is for your target board.
| Pin | Name | Description                                       |
| --- | ---- | ------------------------------------------------- |
| 1   | VCC  | VCC-CN1 with a jumper (Short or Open)             |
| 2   | TX   | TXD-CN1                                           |
| 3   | -    | No connection.                                    |
| 4   | RX   | RXD-CN1                                           |
| 5   | GND  | GND-CN1                                           |
| 6   | I/S  | CTS-CN1 with a jumper (High, Low or No-connect)   |
| 7   | OUT  | DTR/RTS-CN1                                       |
